# HTML to XPath

My Python script to generate XPath queries from a HTML page in less than 100 lines. Specifiy an HTML element's attribute ID to only generate paths to that element and its child elements.

### How to run

Create a virtual enviroment.

```sh
python -m venv env
```

Install dependencies found in `requirements.txt`.

```sh
pip install -r requirements.txt
```

Run main.py

```sh
python main.py
```
